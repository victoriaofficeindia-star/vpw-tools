SIP Calculation Formula
Basic SIP Formula:
FV = P × [((1 + r)^n - 1) / r] × (1 + r)

Where:
FV = Future Value (Maturity Amount)
P = Monthly Investment
r = Monthly Return Rate (Annual Rate ÷ 12 ÷ 100)
n = Number of Months (Years × 12)

With Step-up: Monthly SIP increases by the specified percentage every year, compounding growth.
With Inflation Adjustment: Real value = Maturity Amount ÷ (1 + Inflation Rate)^Years
